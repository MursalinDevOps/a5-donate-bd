What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects.This allows programming languages, especially JavaScript, to manipulate the content, structure, and style of web pages dynamically.

How do select a element from the DOM?

There are multiple ways to select an document from the DOM. The most common methods that i know are shown below :

Get element by Query Selector :

const element = document.getElementById('id_name');

Get elements by Tag name :

const elements = document.getElementsByTagName('tag_name');

Get elements by Class name :

const elements = document.getElementsByClassName('class_name');

Get elements by Query Selector All :

const elements = document.getElementsByQuerySelectorAll('.class_name');

Get element by Query Selector :

const element = document.getElementsByQuerySelector('#id_name');

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of individual child elements. This listener captures events that bubble up from child elements.Event Delegation is Useful for Reducing memory usage and improving performance, Automatically working for dynamically added child elements, as they inherit the event from the parent and Simplifies event handling by centralizing logic in one place.
